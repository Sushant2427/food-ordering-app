# 🍔 KRMUnch - Food Ordering App

This is a Swiggy-style food ordering app built with React and Bootstrap. Converted into an Android app using Capacitor.

## 🧰 Tech Used

- React.js
- Bootstrap
- React Router

- Android Studio
