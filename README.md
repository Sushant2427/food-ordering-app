# 🍔 KRMUnch - Food Ordering App

This is a Swiggy-style food ordering app built using **React.js** and **Bootstrap**. It also includes Android app support using **Capacitor** and **Android Studio**.

---

## 🧰 Tech Stack

- React.js
- Bootstrap
- React Router
- Capacitor
- Android Studio

---
